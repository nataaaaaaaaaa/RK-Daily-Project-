# Daily Project 3 - Sistem Pelacakan Alumni
# Deskripsi

# Sistem Pelacakan Alumni merupakan aplikasi berbasis web yang digunakan untuk menampilkan data alumni serta melakukan pelacakan sederhana terhadap alumni berdasarkan data yang tersimpan di database.

# Aplikasi ini dibuat untuk memenuhi tugas Daily Project 3 pada mata kuliah Rekayasa Perangkat Lunak.

# Teknologi yang Digunakan
# HTML, CSS, JavaScript, Node.js, Express.js, MySQL

# Source Code GitHub
# https://github.com/nataaaaaaaaaa/RK-Daily-Project-.git
